## Toolbar Features
- **Zoom In/Out**: Adjust the zoom level.
- **Search**: Find specific text in the PDF.
- **Annotations**: Highlight, underline, or add notes to the PDF.
- **Bookmarks**: Save specific pages for quick navigation.

## Settings
